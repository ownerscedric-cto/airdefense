# 공기수비대 시공 어시스턴트

새집증후군 "습식 베이크아웃" 시공을 현장에서 진행하면서 1인이 휴대폰/태블릿/노트북으로 띄워두고 쓰는 어시스턴트 웹앱입니다. 백엔드 없이 브라우저 localStorage에 모든 데이터를 저장합니다.

- **작업(Job)** 단위로 고객별 데이터를 관리 — 타임라인·체크리스트·메시지 변수는 작업마다 독립
- **타임라인** — 🛠 **현장용** / 💬 **고객용** 두 갈래
  - 고객용: 카톡 메시지의 시점 기준 진행표
  - 현장용: 방별 동선을 **하위 단계(substeps)**로 체크·메모 (시각·온도 기록)
  - 각각 별도 템플릿 저장·불러오기, 기본 템플릿으로 초기화 지원
- **고객 안내 멘트** — 13종의 카톡 메시지를 변수 치환 후 클립보드로 복사
- **체크리스트** — 단계별 체크 + 공간별 진행 기록표 + 시공 전/후 측정 기록표(환경부 기준치 자동 색상)
- **JSON 내보내기/가져오기** — 기기 간 이전·백업 (v1 → v2 자동 마이그레이션)

모바일 우선 반응형. 다크모드는 OS 설정에 따라 자동 적용됩니다. **PWA**로 동작하므로 핸드폰 홈 화면에 추가해 앱처럼 쓸 수 있고, 한 번 켜본 뒤엔 오프라인에서도 열립니다.

## 핸드폰에서 앱처럼 쓰기

배포된 URL을 모바일 브라우저로 연 뒤:

- **iOS (사파리)**: 공유 버튼 → **홈 화면에 추가**
- **Android (크롬)**: 우상단 메뉴 → **앱 설치** / **홈 화면에 추가**

홈 화면 아이콘으로 열면 주소창 없이 풀스크린으로 동작하고, 사파리의 사이트 데이터 자동 삭제 정책에서도 비교적 안전합니다. 시공이 끝날 때마다 **작업 탭 → JSON 내보내기**로 클라우드(드라이브/iCloud)에 백업해 두면 핸드폰을 갈아도 복구 가능합니다.

## 실행

요구: Node.js 18 이상 (권장: 20+)

```bash
npm install
npm run dev      # http://localhost:5173
```

휴대폰에서도 같은 Wi‑Fi라면 `dev` 실행 시 표시되는 `Network` 주소로 접속해 바로 확인할 수 있습니다.

## 빌드

```bash
npm run build      # dist/ 에 정적 파일 생성
npm run preview    # 빌드 결과 로컬 프리뷰
```

## Vercel 배포

### A. GitHub 연동 (권장)

1. 이 폴더를 GitHub 저장소에 푸시
2. [vercel.com](https://vercel.com) → **New Project** → 해당 저장소 임포트
3. 프레임워크 자동감지(Vite). 그대로 **Deploy**
   - Build Command: `npm run build`
   - Output Directory: `dist`

### B. CLI

```bash
npm i -g vercel
vercel              # 최초 1회 프로젝트 연결
vercel --prod       # 프로덕션 배포
```

## 데이터 보관

- 모든 데이터는 브라우저 localStorage 키 `air-defence:v1` 에 저장됩니다.
- 다른 기기로 옮기려면 **작업 탭 → JSON 내보내기**로 파일을 받아, 새 기기에서 **JSON 가져오기**로 불러오세요. 가져오기는 현재 데이터를 덮어씁니다.
- 브라우저 데이터(쿠키/사이트 데이터) 삭제 시 모든 작업 데이터가 사라집니다.

## 폴더 구조

```
src/
├── App.tsx                # 탭 라우터 + 헤더
├── main.tsx
├── index.css              # Tailwind + 기본 스타일
├── types.ts               # 타입 정의 + 카테고리 색상
├── data/seed.ts           # 시드 데이터 (기본 타임라인 12단계, 멘트 13종, 체크리스트)
├── store/
│   ├── storage.ts         # localStorage 래퍼 (schemaVersion)
│   └── useAppStore.ts     # Context + reducer
├── lib/
│   ├── id.ts              # uid 생성
│   ├── clipboard.ts       # 클립보드 복사 (fallback 포함)
│   └── template.ts        # {{변수}} 치환
├── components/            # 공용 UI (TabNav, Toast, CategoryDot, ConfirmDialog)
└── features/
    ├── jobs/              # 작업 탭 + Import/Export
    ├── timeline/          # 타임라인 탭 + 템플릿 관리
    ├── messages/          # 멘트 탭 + 변수 패널
    └── checklist/         # 체크리스트 탭 + 공간/측정 기록표
```
